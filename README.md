## 🔑 মূল ফিচার ও কম্পোনেন্ট বিবরণ

### 1. **Authentication System** 🔐

#### `src/context/AuthContext.jsx`
- ব্যবহারকারীর লগইন/লজআউট অবস্থা ম্যানেজ করে
- ব্যবহারকারীর তথ্য এবং ভূমিকা (user/admin) সংরক্ষণ করে
- সমস্ত অ্যাপে উপলব্ধ থাকে

#### `src/utils/providers/AuthProvider.jsx`
প্রধান ফাংশন:
- `signUp()` - নতুন ব্যবহারকারী তৈরি
- `signIn()` - লগইন (মনে রাখুন বিকল্প)
- `googleSignIn()` - Google দ্বারা লগইন
- `logout()` - লগআউট
- `passwordReset()` - পাসওয়ার্ড রিসেট ইমেইল পাঠানো
- `updateUserProfile()` - ���্যবহারকারীর প্রোফাইল আপডেট করা

#### `src/utils/Private/PrivateRoute.jsx`
- সুরক্ষিত রুট প্রোটেক্ট করে
- অনথেন্টিকেটেড ব্যবহারকারীদের লগইন পেজে পুনঃনির্দেশ করে

### 2. **Cart Management** 🛒

#### `src/context/CartContext.jsx`
প্রধান ফাংশন:
- `addToCart()` - পণ্য কার্টে যোগ করা
- `removeFromCart()` - পণ্য কার্ট থেকে সরানো
- `updateQuantity()` - পণ্যের পরিমাণ পরিবর্তন
- `clearCart()` - সম্পূর্ণ কার্ট খালি করা
- `cartItems` - বর্তমান কার্ট আইটেমস অ্যাক্সেস করা

## 🔄 ডাটা ফ্লো এবং পারস্পরিক সম্পর্ক

```
┌─────────────────────────────────────────────┐
│         Firebase Authentication              │
│  (Email/Password, Google Login)             │
└──────────────┬──────────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────────┐
│      AuthProvider + AuthContext              │
│  (Manages user state globally)              │
└──────────┬────────────────┬────────────────┘
           │                │
           ▼                ▼
    ┌────────────┐  ┌──────────────────┐
    │ SignUp.jsx │  │  SignIn.jsx      │
    └────────────┘  └──────────────────┘
           │                │
           └────────┬───────┘
                    ▼
    ┌─────────────────────────────────┐
    │   Home.jsx + All Components      │
    │  (Access user from context)     │
    └────────┬────────────────────────┘
             │
             ▼
    ┌─────────────────────────────┐
    │  CartContext + CartProvider  │
    │  (Manages shopping cart)    │
    └────────┬────────────────────┘
             │
             ▼
    ┌──────────────────────────┐
    │  Checkout → Payment      │
    │  (Process order)         │
    └────────┬─────────────────┘
             │
             ▼
    ┌──────────────────────────┐
    │  OrderConfirmation       │
    │  (Clear cart, show ID)   │
    └──────────────────────────┘
```

### **ডাটা ফ্লো ব্যাখ্যা:**

1. **অথেন্টিকেশন**: ব্যবহারকারী Firebase দ্বারা সাইন আপ/সাইন ইন করে
2. **স্টেট ম্যানেজমেন্ট**: AuthContext সমস্ত অ্যাপে ব্যবহারকারীর তথ্য সরবরাহ করে
3. **শপিং**: ব্যবহারকারী পণ্য ব্রাউজ করে এবং CartContext এ যোগ করে
4. **চেকআউট**: কার্টের আইটেমগুলি দেখায় এবং পেমেন্টে যায়
5. **অর্ডার প্লেসমেন্ট**: পেমেন্ট ফর্ম জমা দেওয়ার পর অর্ডার কনফার্মেশন পেজ দেখা যায়
6. **ট্র্যাকিং**: ব্যবহারকারী অর্ডার ট্র্যাকিং পেজে অর্ডার ID দিয়ে সার্চ করে স্ট্যাটাস দেখতে পারে
